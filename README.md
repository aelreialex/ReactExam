# Motion UI
### Använder motion ui mest för att lägga till lite liv när man klickar sig runt på sidor samt knappar med onhover effekter och onclick effekter
### Motion UI är ett tillägg som gör det lätt att lägga till animationer till olika element och layouts
### Tyckte att jag behövde lite liv i sidan för att få den att ha mer känsla så att den inte är för platt

# Swiper
### Använder Swiper enbart för att kunna rotera runt biljetterna man har köpt på sista sidan.
### Swiper är typ som motion egentligen finns card effekter i motion också men denna är mer enbart för slide content.
### Följde skissen behövde ha det här för kortdelen.

# Toast
### Ger en visuell bekräftelse när man lägger till en biljett i varukorgen
### Ger en liten visuell uppdatering som man kan sätta på knappar eller lyssnare av något slag. En liten popup som säger "Bra jobbat!" eller "Skiten gick fel!" beroende på vad för element man ger den som .sucess exempelvis.
### Roligt att få upp något på skärmen som en bekräftelse skapar liv i sidan.
